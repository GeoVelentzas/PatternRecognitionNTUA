%%%%%%%%%%%%Feature Transcription%%%%%%%%%%%
Each row corresponds to the features extracted from one frame of the video.
The format in each row is comma separated in other works there are 7 fields that are separated using commas (','). The fields correspond to :
1) Frame Number
2) Head Centroid x coordinate
3) Head Centroid y coordinate
4) Right Hand Centroid x coordinate
5) Right Hand Centroid y coordinate
6) Left  Hand Centroid x coordinate
7) Left Hand  Centroid y coordinate


ps. (1,1) coordinate correspond to the upper left corner of the image.
